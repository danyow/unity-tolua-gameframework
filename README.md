# Unity-Lua-Framework

#### 介绍

- 基于toLua扩展的Unity热跟新实用框架，完全继承MonoBehaviour生命周期，开箱即用。

- 组件内控制AssetBundle回收，内存管理你说了算。

- 自带Tween动画模块，一行代码实现动画，比DotTween、LeanTween等四大Tween工具更加灵活轻便。

- 更多功能不断完善中。。
